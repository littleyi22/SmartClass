# SmartClass 智慧教室助手 v.4

這是一個為老師設計的互動式智慧教室好幫手，專為管理班級、隨機抽籤、搶答活動以及課堂活動所打造。

## 功能特色
- **主控面板**: 總覽全班人數、優良表現與未交功課的人數統計。同時提供教學進度紀錄與今日功課摘要。
- **班級狀態**: 學生資料查詢、Excel名單匯入、分數管理功能。
- **隨機分組**: 支援自動隨機分組、手動微調、抽小組及抽組員。
- **課堂神器**:
  - 幸運大抽籤 (隨機挑選學生)
  - 超級搶答王 (學生掃碼同步搶答)
  - 專業計時器 (可用於分組討論或測驗倒數)
  - 分貝監控器 (幫助維持班級秩序)
  - 簡易白板 (隨手繪圖、重點紀事)
  - 投擲骰子 (隨機點數與機率展示)
- **今日功課板**: 編輯功課並即時顯示給學生看。
- **系統設置**: 提供不同介面風格主題設定，支援多班級管理以及 Google 帳號授權將資料同步至雲端。

## 技術與資源
- **前端開發**: HTML, CSS, JavaScript
- **第三方函式庫**:
  - Lucide Icons (圖示庫)
  - Canvas Confetti (彩帶特效)
  - QRCode.js (產出 QR Code)
  - SheetJS (讀取 Excel 匯入學生清單)
  - Google API (GIS & Client Library，供雲端存檔功能使用)
