# SmartClass 智慧教室助手 v.5.5

這是一個為老師設計的互動式智慧教室好幫手，專為管理班級、隨機抽籤、搶答活動以及課堂活動所打造。

## 功能特色
- **主控面板**: 總覽全班人數、優良表現與未交功課的人數統計。同時提供教學進度紀錄與今日功課摘要。
- **班級狀態**: 學生資料查詢、Excel名單匯入、分數管理功能。
- **點名簿**: 記錄上學簽到(含遲到判定)、刷牙及兩組自訂標籤，支援一鍵匯出 Excel 功能。
- **隨機分組**: 支援自動隨機分組、手動微調、抽小組及抽組員。連動分數可自訂開關。
- **課堂神器**:
  - 幸運大抽籤 (隨機挑選學生)
  - 超級搶答王 (學生掃碼同步搶答)
  - 專業計時器 (可用於分組討論或測驗倒數)
  - 分貝監控器 (幫助維持班級秩序)
  - 簡易白板 (隨手繪圖、重點紀事)
  - 投擲骰子 (隨機點數與機率展示)
  - 廣播網址 (一鍵推播連結)
  - 隨堂測驗 (簡答與單選畫板)
  - 收取短文 (收集全班即時文字)
  - 收取檔案 (專屬學生上傳通道)
- **系統設置**: 提供多變主題選擇(包含馬卡龍色系)，並支援 Google 帳號授權將資料一鍵同步至雲端硬碟。

## 技術與資源
- **前端開發**: HTML, CSS, JavaScript
- **第三方函式庫**:
  - Lucide Icons (圖示庫)
  - Canvas Confetti (彩帶特效)
  - QRCode.js (產出 QR Code)
  - SheetJS (讀取 Excel 及匯外學生與點名清單)
  - PeerJS (提供 P2P WebRTC 機制，負責學生端即時連線互動)
  - Google API (GIS & Client Library，供雲端存檔功能使用)
